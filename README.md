## 📖 简介

Obsidian-Toolkit 是一个专注于 Obsidian 使用的知识库，收录了各类实用工具、插件推荐、使用技巧和问题解决方案。无论你是 Obsidian 新手还是资深用户，都能从这里找到有用的资源。

## ✨ 功能特性

- 📦 **插件推荐** - 精选必备和实用的 Obsidian 插件
- 🌐 **浏览器扩展** - 配合 Obsidian 使用的 Chrome 插件
- 📊 **Dataview 查询** - 实用的数据筛查和看板配置示例
- 📝 **Web Clipper 模板** - 网页剪藏模板集合
- 🔧 **问题排查** - 常见问题和解决方案文档
- 🎨 **模板和脚本** - QuickAdd 模板和 JavaScript 脚本

## 🚀 快速开始

### Obsidian 插件推荐

查看我们精选的插件列表，包括必备插件和精品插件：

- [Obsidian 常用插件列表](./Obsidian-常用插件列表.md)

**必备插件包括：**
- **Obsidian-git** - Git 版本控制和自动备份
- **Obsidian-dataview** - 强大的数据查询和筛选工具
- **nldates-obsidian** - 自然语言日期输入
- **obsidian-auto-link-title** - 自动获取URL链接标题

**精品插件包括：**
- **remotely-save** - 第三方云同步插件
- **longform** - 长文写作工具
